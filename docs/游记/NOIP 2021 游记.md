### T1 [报数](https://www.luogu.com.cn/problem/P7960) (100' $\rightarrow$ 0')

一看到 T1，难道不就是埃氏筛的变形？

二话不说，写好了，大约 5 分钟。

然而觉得不够快，想要再优化一点， `cin` 和 `cout` 换成了 `scanf` 和 `printf`（或 `puts`）。

写下了这样的代码。

```cpp
#include<stdio.h>
int v[10003007];
int nums[800000], cnt;
int t, x, u;
int p(int x)
{
    while(x)
        if(x%10==7)
            return 1;
        else
            x/=10;
    return 0;
}
int main()
{
    for(int i=1;i<=10003000;i++)
        if(p(i))
        {
            for(int j=1;i*j<=10003000;j++)
                v[i*j]=-1;
        }
    for(int i=1;i<=10003000;i++)
        if(v[i]==0)
            nums[++cnt]=i, v[i]=cnt;
    scanf("%d",&t);
    while(t--)
    {
        scanf("%d",&x);
        if(v[x]==-1)
            puts("-1\n");
        else
            printf("%d\n",nums[v[x]+1]);
    }
}
```

然而很不幸，问题就出在 `puts` 上。

`puts` 和 Python 的 `print` 很类似，每输出一次就换一次行，所以我的 `puts("-1\n")` 的 `\n`，属于画蛇添足。

很不幸爆零了。
